# Sıra

**Turn your pre-launch list into a growth loop.** Sıra collects signups, ranks them by referrals, and rewards your biggest fans — so the list grows itself before you've shipped.

```bash
npm install
npm run dev          # → http://localhost:3000  (demo mode, no keys needed)
```

Open in **Claude Code** and say **"set up this project"** (or run **`/setup`**) to wire your brand plus **Resend** (emails) and **Supabase** (signups) keys. By hand? See [`SETUP.md`](./SETUP.md).

Pages: `Waitlist` (dashboard) · `Referrals` (leaderboard + viral coefficient) · `Emails` (lifecycle templates) · `Embed` (copy-paste widget) · `Settings`. Built on the GoatStarter template — Next.js 16 · React 19 · Tailwind v4.
