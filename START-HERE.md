# 👋 Start here / Buradan başla

This folder is a **complete app** — it already runs with realistic demo data and
needs **zero setup to preview**. When you're ready to make it yours and put it
online, an AI assistant (Claude Code) walks you through everything.

Bu klasör **çalışan bir uygulama** — gerçekçi demo verisiyle birlikte gelir ve
önizlemek için **hiçbir kurulum gerektirmez**. Kendine ait yapıp internete çıkmaya
hazır olduğunda, bir yapay zekâ asistanı (Claude Code) her şeyi sana adım adım
anlatır.

---

## 1) Preview it now / Hemen önizle

```bash
npm install
npm run dev      # → http://localhost:3000
```

## 2) Make it yours / Kendine ait yap

Open this folder in **Claude Code** and say:

> **"set up this project"**  ·  **"bu projeyi kur"**

Claude will open the **visual guide** in `setup-guide/` and walk you through it
**step by step**, asking only for what each step needs:

| Step | Görsel | What happens |
|---|---|---|
| 0 · Overview | `setup-guide/0-overview.png` | The whole flow at a glance |
| 1 · Build | `setup-guide/1-build-with-claude.png` | Brand it: name, logo, colors, copy (Claude edits the code) |
| 2 · Supabase | `setup-guide/2-connect-supabase.png` | Free backend: database + login |
| 3 · GitHub | `setup-guide/3-push-to-github.png` | Save your code in the cloud |
| 4 · Vercel | `setup-guide/4-deploy-to-vercel.png` | Deploy — your app goes live |
| 5 · Stripe | (in overview) | Take payments |

You don't need to know what these are — Claude explains each one and asks you
the right questions. Bunların ne olduğunu bilmene gerek yok — Claude her birini
açıklar ve sana doğru soruları sorar.
